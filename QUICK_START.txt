🎯 OPSI 1: TERMUDAH (Gunakan Script)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Buka File Explorer
2. Pergi ke: C:\laragon\www\backend-absensi\
3. DOUBLE-CLICK file: START.bat
4. Tunggu 10 detik sampai 2 terminal terbuka
5. Buka Browser → http://localhost:5173/
6. LOGIN: admin / 123456

SELESAI! ✅


🎯 OPSI 2: MANUAL (Jika Script tidak work)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TERMINAL 1 (JANGAN TUTUP):
─────────────────────────
1. Buka Command Prompt (Tekan: Win + R, ketik "cmd", Enter)
2. Copy-paste perintah ini:
   
   cd c:\laragon\www\backend-absensi && node index.js

3. Tunggu sampai muncul tulisan:
   "Server jalan di http://localhost:5000 ✅"


TERMINAL 2 (BUKA BARU):
──────────────────────
1. Buka Command Prompt BARU lagi (Tekan: Win + R, ketik "cmd", Enter)
2. Copy-paste perintah ini:

   cd c:\laragon\www\backend-absensi\frontend-absensi && npm run dev

3. Tunggu sampai muncul tulisan:
   "Local: http://localhost:5173/"


BROWSER:
────────
1. Buka browser (Chrome/Firefox)
2. Ketik: http://localhost:5173/
3. LOGIN dengan:
   Username: admin
   Password: 123456

🔴 MASALAH UMUM
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "ECONNREFUSED port 3306"
   → MySQL tidak running
   → Solusi: Buka Laragon, klik "Start All"

❌ "Cannot GET /"
   → Akses localhost:5000 di browser (salah!)
   → Solusi: Akses http://localhost:5173/ (yang benar)

❌ "Port 5173 already in use"
   → npm masih running dari sebelumnya
   → Solusi: Buka Task Manager, kill npm, jalankan lagi

❌ Login tidak berhasil / error di Chrome console
   → Backend tidak running
   → Solusi: Check Terminal 1, pastikan ada tulisan "Server jalan"

❌ Frontend blank/loading terus
   → Mungkin node_modules belum install
   → Solusi: Di Terminal 2, Ctrl+C, kemudian:
      npm install
      npm run dev
